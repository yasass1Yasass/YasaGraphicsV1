# EcoVenture Starter (React + TypeScript + Tailwind + React Router)

## Quick start
```bash
npm install
npm run dev
```

## Stack
- Vite
- React + TypeScript
- TailwindCSS
- React Router DOM

## Tailwind
Configured with `tailwind.config.js` and `postcss.config.js`.
Utilities are loaded via `src/index.css`:
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
